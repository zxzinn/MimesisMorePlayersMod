## ğŸ› Troubleshooting

### Game crashes / Mod not loading?

1. Verify MelonLoader is installed
2. Check `MorePlayers.dll` is in `Mods` folder
3. Check `MelonLoader/Latest.log` for errors
4. Report issue at [Issues](https://github.com/zxzinn/MimesisMorePlayersMod/issues)

### Still limited to 4 players?

- Verify **host** has mod installed
- Check mod is loaded (see game log)
- May be network/Steam connection limits
